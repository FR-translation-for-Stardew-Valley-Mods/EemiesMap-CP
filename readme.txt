Eemie's Just A New Map Recolour (Content Patcher Edition) - Version 1.7.0
An official conversion of Eemie's map recolor for Content Patcher.

:: Required Mods::
SMAPI (https://smapi.io/)
Content Patcher (https://www.nexusmods.com/stardewvalley/mods/1915)

:: What This Mod Changes::
-Desaturates much of the outdoor palettes
-New sprites for all three regular trees
-New grass sprites
-Includes both Eemie's grass and strawberrymilk95's grass, for a total of six grass options.
-Includes zanderb14's ladder recolors for compatibility with Expanded Maps, etc, as well as his world map recolor.  Now seasonal!
-Now includes beach and desert recolors

:: Installation Instructions::
1. Download and install the latest version of SMAPI
2. Download and install the latest version of Content Patcher
3. Copy '[CP] Eemie's Map Recolors' folder to Mods
4. Run the game once to generate the config file, options below.
5. You're done! Run the game.

:: Config Options::
-SpringColor - choose between "Pink" and "Green"; defaults to "Green"
-FallColor - choose between "Green" and "Orange"; defaults to "Green"
-DirtColor - choose between "Classic" and "Light"; defaults to "Classic"
-UseFallBushes - choose if you want green or red bushes in fall; uses true/false, defaults to "true"
-UseEemiePine - choose if you want to use Eemie's birch tree or the vanilla pine tree.  True/False option, defaults to "true".
-GrassVariant- allows choices between each of Ali's five grass variants or "None" if you wish to use Eemie's grass or a different grass altogether.  Defaults to "None".  Variants as follows:
--"Eemie"
--"AliEemie"
--"AliNormal"
--"AliAlternate"
--"AliVibrant"
--"AliAlternateVibrant"
-UseWorldMapRecolor - choose if you want to use Zander's world map recolor with True/False.  Defaults to "true".  If you use NPC Map Locations by Bouhm, that mod includes a compatible map by default; this mod will automatically disable the edit if Bouhm's mod is detected.
-Petbowl - used for custom farm maps that don't use the wooden pet area.  Defaults to the vanilla option. Options are as follows:
--"Vanilla"
--"Void"
--"Dirt"
--"Grass"
--"Darkgrass"
